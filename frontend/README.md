# PDF Tools (Client-Side)

This directory contains a client-side implementation of the PDF tools using React and pdf-lib.
No Python server is required.

## How to Run

1.  Simply open `index.html` in your web browser.
2.  Or host it on GitHub Pages.

## Dependencies (CDN)

- React, ReactDOM
- Tailwind CSS
- pdf-lib (PDF Manipulation)
- popdfjs-dist (PDF Rendering)
- JSZip (Zip creation)
- Lucide (Icons)

## Features

- **Split**: All pages or Custom range.
- **Merge**: Combine multiple PDFs.
- **Reorder**: Drag and drop page ordering.
- **PDF -> Image**: Convert PDF pages to JPEGs (ZIP).
- **Image -> PDF**: Convert Images to PDF.
- **Protect**: Add password.
- **N-up**: 4-in-1 page aggregation.
